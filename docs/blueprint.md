# **App Name**: REON AI Prep

## Core Features:

- AI Study Plan Generation: Generates personalized study plans based on user's target exam, weak subjects, and available hours using generative AI to produce reasoning to create the plan, using previous performance as a tool.
- AI Doubt Solving: Allows users to ask questions about any topic and receive instant answers from LIBRA, an AI-powered chatbot that works as a tool.
- Comprehensive Dashboard: Displays user's overall progress, recent results, active missions, and time spent analysis in a clear and concise manner.
- Multi-Language Support: Provides content and interface in multiple languages including Telugu, Hindi, Tamil, and English.
- Utilities Dock: Offers quick access to utilities such as Alarm, Stopwatch, Calculator, and Calendar in a compact dock.
- Smart Notes: Allow the user to upload any PDF, which is then converted into interactive flashcards using AI that works as a tool. The AI can provide mnemonics and highlight sections of the document as well.
- Leaderboard: Displays user's ranking among other aspirants and tracks recent games and progress.

## Style Guidelines:

- Primary color: Dark blue (#24292E) to evoke trust and professionalism.
- Background color: Light gray (#F5F5F5) for a clean and modern feel.
- Accent color: Light Blue (#ADD8E6) for interactive elements and highlights.
- Body font: 'Inter' sans-serif for a modern, machined look and readability.
- Headline font: 'Space Grotesk' sans-serif for computerized, techy feel in headlines. (Use 'Inter' for body).
- Use simple, line-based icons for a clean and consistent look.
- Use subtle animations and transitions to provide feedback and enhance user experience.
